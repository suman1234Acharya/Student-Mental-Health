# Student-Mental-Health
🧠 Project Title: Student Mental Health Analysis

This project is about analyzing students’ mental health using machine learning techniques. It helps in predicting issues like depression, anxiety, and panic attacks.

📌 Overview

This section explains the main purpose of the project:

To study student mental health
To use data and ML models for prediction
To identify students who may need help early
📊 Dataset Features

This part describes what data is used in the project.
Each column (feature) represents student information:

Age → Student’s age
Gender → Male/Female
Course → Field of study
Year of Study → Academic level
CGPA → Academic performance
Marital Status → Personal status
Depression / Anxiety / Panic Attack → Mental health condition
Treatment → Whether the student took help

👉 These features are used to train the ML model.

🎯 Project Goals

This explains what you want to achieve:

Find patterns in student mental health
Build prediction models
Help in early detection of problems
⚙️ Workflow

This shows step-by-step process:

Data Collection → Get dataset
Data Cleaning → Remove errors, missing values
EDA (Exploratory Data Analysis) → Understand data using graphs
Feature Engineering → Prepare data for ML
Model Training → Train ML models
Evaluation → Check accuracy
🤖 Models Used

Different machine learning models are used:

Logistic Regression → Basic model
Random Forest → Gives high accuracy
Decision Tree → Easy to understand
SVM → Good for classification
KNN → Uses similarity between data
📈 Results

This section explains the outcome:

Model gives good accuracy
Random Forest performs best
Important factors affecting mental health are identified
